The Index Diachronica Database

collection of historical sound changes

The Index Diachronica is a giant catalog of sound changes.

structured dataset

By transforming this catalog into machine readable files of structured data, we will be able to do a lot of cool things with it.

human editable

The basis of this dataset build easy to understand file formats such as toml.

scientific basis

But that's not everything: We also want to improve the rigorosity of the dataset. For example, the Glottolog codes are used instead of family names

wider coverage

We also want to expand the number of covered language families.

friendly community

I would like this project to be an open and welcoming one
